How to Install Photoshop Brushes

  1. Go to the Presets Manager (Edit > Presets > Preset Manager or Edit > Preset Manager).
  2. In the Preset Type drop-down menu, select Brushes.
  3. Click on the Load button, browse for the .atn file, then click OK.

Note: You will need Photoshop CS or newer to use these brushes. 5000px brushes are only supported in Photoshop CS6 and newer.
