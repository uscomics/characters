Getting Started


How to use the Styles:

Open a PSD file of your choice.

Change to 300Dpi:
Go to - Image - Image Size - and change 72 to 300Dpi and click OK
Now doubleclick the Smart Object (Main Layer) and change the PSB file of the Smart Object to 300Dpi also!

Open a PSD file of your choice.

Change the text and font to your own:

Double click on the thumb of the RED layer. (Or Right click on the layer and choose Edit Contents)
In this new window change the text and the font add shapes if you want.
- Make sure that all the colors are white!
Now save the new window with Ctrl + s and close it. Voila, your 3D style is ready!



Change the size:

Making the text bigger: Just change the size of the font.
Making the text smaller: Select all layers except the Background layers (BG) and right click on a layer and choose "Convert to Smart Object" than resize the text with Free Transform (Ctrl+T)


Change the angle of the 3D:

When your in the smart object move the text or shape up or down in the window to change the angle of the 3D.
