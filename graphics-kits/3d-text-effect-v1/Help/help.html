Getting Started


How to use the Styles:

Open a PSD file of your choice.

Change the text and font to your own:

Double click on the thumb of the RED layer. (Or Right click on the layer and choose Edit Contents)
In this new window change the text and the font add shapes if you want.
Change the Text, Font and Color to your own save the new window with Ctrl + s and close it. Voila, your 3D style is ready!



Change the size:

Making the text bigger or smaller: Just change the size of the font or go to "Image - Image Size" and change the size.

Change the Color

You can change the color by simply change the color of the font in the Character Panel after you opened the Smart Object.
